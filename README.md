# YouTube AI Studio — استوديو يوتيوب الذكي 🎬🤖

منصة ويب متكاملة (Frontend فقط — بدون خادم) لإدارة وتنمية قنوات يوتيوب بالذكاء الاصطناعي، تضم **147 أداة احترافية** موزعة على 9 وحدات، بواجهة عربية RTL كاملة مع وضع ليلي، وأوركسترا نماذج تجمع Gemini و OpenAI GPT بـ 4 أوضاع عمل.

### أوركسترا النماذج (جديد — طبقة `AI` فى api.js)
- **نماذج OpenAI GPT** بجانب Gemini: gpt-5.5 / gpt-5.5-mini / gpt-5.5-nano / gpt-5 / gpt-5-mini / gpt-5-nano (مفتاح OpenAI اختيارى فى الإعدادات، مع دعم الصور وJSON).
- **4 أوضاع عمل** تُختار من الإعدادات وتسرى على كل الأدوات:
  1. نموذج واحد (افتراضى)
  2. نموذجان بالتوازى وعرض النتيجتين معاً
  3. نموذجان + حَكَم ثالث يدمج أفضل ما فيهما
  4. مناقشة: مسودة A ← نقد B ← صياغة نهائية A
- **اختيار النموذج A / B / الحَكَم** من قوائم تجمع عائلتى Gemini وGPT، مع تحويل تلقائى لـ Gemini عند فشل GPT.
- **حقن تاريخ اليوم الفعلى** تلقائياً فى كل استدعاء AI حتى لا تُعامل بيانات القناة الحديثة كتواريخ مستقبلية.

### البنية التحتية
- **محرك أدوات موحد** (`engine.js`): يبنى صفحة كل أداة من تعريفها، يجمع المدخلات، يعرض التقدم، يعالج الأخطاء برسائل مفيدة، ويحفظ النتائج تلقائياً.
- **وضع المحادثة**: أداتا محادثة كاملتان (المدرب الذكي + Copilot) بذاكرة محفوظة وسياق القناة.
- **تكامل Gemini API** مع تبديل تلقائى بين 6 نماذج عند الفشل + توليد JSON منظم + بحث إنترنت حى (Google Search Grounding).
- **تكامل YouTube Data API v3**: قنوات، فيديوهات، تعليقات، بحث، قوائم — مع كاش ذكى (10-60 دقيقة) لتوفير الكوتا.
- **كتابة فعلية على يوتيوب** (جديد): `videos.update` لتعديل العنوان/الوصف/الوسوم + `thumbnails.set` لرفع ثمبنيل — عبر OAuth بصلاحية `youtube.force-ssl` (طبقة `GAuth` لإدارة رموز كل صلاحية).
- **تكامل YouTube Analytics API** (اختيارى عبر OAuth): CTR، وقت المشاهدة، الإيرادات — يتدهور بأمان لو غير متصل.

### أدوات بارزة جديدة
- **رادار الترندات** (`#/tool/trends-radar` — m4): بحث حى فى الإنترنت (Google Search Grounding) + مسح يوتيوب لأحدث فيديوهات النيتش ← خريطة ترندات + 10 أفكار فيديوهات لركوب الموجة.
- **محرر معلومات الفيديو** (`#/tool/video-info-editor` — m2): عرض وتقييم العنوان/الوصف/الوسوم بدرجات، اقتراحات ذكية، حقول تعديل + رفع ثمبنيل جديد، تحليل قبل/بعد، وزر تطبيق التعديلات فعلياً على الفيديو (يحافظ على categoryId الأصلى).
- **برومبت توليد الثمبنيل** المحدَّث: يولّد إضافياً **برومبت تخطيط (Layout Prompt)** يصف مواضع العناصر داخل الإطار لاستخدامه كمرجع مع البرومبت الرئيسى.
- **ذاكرة النظام**: قاعدة معرفة القناة (ملف، جمهور، أسلوب، قواعد، أنماط رابحة) تُحقن تلقائياً فى كل أدوات AI.
- **سجل النتائج + سجل التوصيات + سجل النشاط (Audit)** — كلها محلية مع تصدير/استيراد نسخة احتياطية كاملة.
- **تصدير النتائج**: نسخ / TXT / MD / JSON / PDF (عبر الطباعة).
- **رسوم بيانية** Chart.js، حلقات درجات، أشرطة تقدم، جداول بيانات.
- واجهة عربية RTL + وضع ليلي + استجابة كاملة للموبايل + بحث فورى فى الأدوات.

## 🔗 مداخل الاستخدام (Hash Routes)
- `#/` — الرئيسية (بطاقات سريعة + الوحدات + آخر النتائج)
- `#/module/{m1..m9}` — شبكة أدوات الوحدة
- `#/tool/{tool-id}` — صفحة أداة (مثال: `#/tool/channel-dashboard`، `#/tool/ai-workspace`)
- `#/settings` — الإعدادات (المفاتيح، القناة، النسخ الاحتياطى)
- `#/history` — سجل النتائج

## ⚙️ الإعداد المطلوب من المستخدم
1. **مفتاح Gemini API** (مجانى): من [Google AI Studio](https://aistudio.google.com/apikey)
2. **مفتاح YouTube Data API v3**: من [Google Cloud Console](https://console.cloud.google.com/apis/credentials) بعد تفعيل الخدمة
3. (اختيارى) **مفتاح OpenAI API** لتفعيل نماذج GPT: من [OpenAI Platform](https://platform.openai.com/api-keys)
4. (اختيارى) **OAuth Client ID** لتفعيل Analytics الخاصة وتعديل الفيديوهات — أضف نطاق الموقع فى Authorized JavaScript origins
5. (اختيارى) اختر **وضع أوركسترا النماذج** والنموذج A/B/الحَكَم من الإعدادات
6. احفظ الكل فى صفحة الإعدادات — كل شىء يُخزن محلياً فى المتصفح فقط.

## 💾 التخزين
- كل البيانات فى `localStorage` بادئة `yais_` (لا خوادم، لا قواعد بيانات خارجية).
- الطلبات تذهب مباشرة من المتصفح إلى Google APIs فقط.

## ❌ غير منفذ بعد / حدود معروفة
- تعديل الفيديوهات الفعلى يعمل فقط لفيديوهات حسابك أنت (ويتطلب OAuth Client ID + موافقة صاحب القناة عند كل جلسة) — رفع فيديوهات جديدة غير مدعوم.
- التنبيهات "المجدولة" تعمل عند فتح الأداة (لا يوجد خادم للتشغيل الدورى).
- توليد صور الثمبنيل غير متاح (تحليل الثمبنيل بالصورة متاح، وبرومبت التوليد + برومبت التخطيط جاهزان لأدوات خارجية).
- نماذج GPT تتطلب مفتاح OpenAI مدفوعاً؛ بدونه تعمل كل الأدوات بنماذج Gemini.
