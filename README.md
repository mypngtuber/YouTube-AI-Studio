# YouTube AI Studio — استوديو يوتيوب الذكي 🎬🤖

منصة ويب متكاملة (Frontend فقط — بدون خادم) لإدارة وتنمية قنوات يوتيوب بالذكاء الاصطناعي، تضم **145 أداة احترافية** موزعة على 9 وحدات، بواجهة عربية RTL كاملة مع وضع ليلي.

## ✅ الميزات المكتملة

|--------|-------|-------------|
| m1 | لوحة التحكم والتقارير | 10 |
| m2 | إدارة الفيديوهات | 10 |
| m3 | التحليلات | 11 |

### البنية التحتية
- **محرك أدوات موحد** (`engine.js`): يبنى صفحة كل أداة من تعريفها، يجمع المدخلات، يعرض التقدم، يعالج الأخطاء برسائل مفيدة، ويحفظ النتائج تلقائياً.
- **وضع المحادثة**: أداتا محادثة كاملتان (المدرب الذكي + Copilot) بذاكرة محفوظة وسياق القناة.
- **تكامل Gemini API** مع تبديل تلقائى بين 6 نماذج عند الفشل + توليد JSON منظم.
- **تكامل YouTube Data API v3**: قنوات، فيديوهات، تعليقات، بحث، قوائم — مع كاش ذكى (10-60 دقيقة) لتوفير الكوتا.
- **تكامل YouTube Analytics API** (اختيارى عبر OAuth): CTR، وقت المشاهدة، الإيرادات — يتدهور بأمان لو غير متصل.
- **ذاكرة النظام**: قاعدة معرفة القناة (ملف، جمهور، أسلوب، قواعد، أنماط رابحة) تُحقن تلقائياً فى كل أدوات AI.
- **سجل النتائج + سجل التوصيات + سجل النشاط (Audit)** — كلها محلية مع تصدير/استيراد نسخة احتياطية كاملة.
- **تصدير النتائج**: نسخ / TXT / MD / JSON / PDF (عبر الطباعة).
- **رسوم بيانية** Chart.js، حلقات درجات، أشرطة تقدم، جداول بيانات.
- واجهة عربية RTL + وضع ليلي + استجابة كاملة للموبايل + بحث فورى فى الأدوات.

## 🔗 مداخل الاستخدام (Hash Routes)
- `#/` — الرئيسية (بطاقات سريعة + الوحدات + آخر النتائج)
- `#/module/{m1..m9}` — شبكة أدوات الوحدة
- `#/tool/{tool-id}` — صفحة أداة (مثال: `#/tool/channel-dashboard`، `#/tool/ai-workspace`)
- `#/settings` — الإعدادات (المفاتيح، القناة، النسخ الاحتياطى)
- `#/history` — سجل النتائج

## ⚙️ الإعداد المطلوب من المستخدم
1. **مفتاح Gemini API** (مجانى): من [Google AI Studio](https://aistudio.google.com/apikey)
2. **مفتاح YouTube Data API v3**: من [Google Cloud Console](https://console.cloud.google.com/apis/credentials) بعد تفعيل الخدمة
3. (اختيارى) **OAuth Client ID** لتفعيل Analytics الخاصة — أضف نطاق الموقع فى Authorized JavaScript origins
4. احفظ الكل فى صفحة الإعدادات — كل شىء يُخزن محلياً فى المتصفح فقط.

## 💾 التخزين
- كل البيانات فى `localStorage` بادئة `yais_` (لا خوادم، لا قواعد بيانات خارجية).
- الطلبات تذهب مباشرة من المتصفح إلى Google APIs فقط.

## ❌ غير منفذ بعد / حدود معروفة
- لا رفع/تعديل فعلى لفيديوهات يوتيوب (يتطلب OAuth بنطاق كتابة + موافقة Google) — الأدوات تنتج المحتوى للنسخ واللصق.
- التنبيهات "المجدولة" تعمل عند فتح الأداة (لا يوجد خادم للتشغيل الدورى).
- توليد صور الثمبنيل غير متاح (تحليل الثمبنيل بالصورة متاح) — Gemini API النصى فقط.

## 🔮 الخطوات التالية المقترحة
